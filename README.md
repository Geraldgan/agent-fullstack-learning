# Agent Fullstack Learning

面向 **iOS 开发工程师转 Agent 全栈工程师** 的学习资料仓库。

适合这样的你：

- 会 Objective-C / Swift
- 有客户端开发经验
- 大学学过 Java，但后端实战偏少
- 想补前端、Java 后端、Agent 工程能力
- 希望从“会调模型 API”进阶到“能独立落地 Agent 产品”

这个仓库的目标不是只给你一个学习计划，而是尽量同时提供：

- **路线图**：知道该学什么
- **教学文档**：知道每个知识点是什么、为什么、怎么用
- **速查手册**：做项目时随时翻
- **练习题**：逼自己动手
- **项目设计资料**：把知识串成真实项目

# 这套资料适合什么人

适合：

- iOS / Android 客户端工程师转全栈
- 有 Java 基础但缺少实战的人
- 想从传统开发转向 AI / Agent 应用开发的人
- 想做 Next.js + Spring Boot + Agent 项目的人

不太适合：

- 想学特别底层的编译原理、JVM 深水区、React Fiber 源码的人
- 只想刷八股、不准备做项目的人

这个仓库更偏向：

**先做出东西，再逐步补系统性。**

---

# 你最终应该达到的能力

如果你把这套资料真正学完并动手实践，理想状态下你会逐步具备这些能力：

- 能写一个 Next.js 前端
- 能写一个 Spring Boot 后端
- 能设计用户 / 会话 / 消息表
- 能完成前后端联调
- 能接入大模型 API
- 能理解 Tool Calling / RAG / Workflow 的工程落地方式
- 能做出一个 Agent Chat 类项目原型

# 学习建议

给你一个最实用的建议：

不要把这些资料当“收藏夹”，要把它们当“施工图”。

比较合理的方式是：

1. 先读一份总览
2. 再读一份教学文档
3. 一边做练习一边翻速查手册
4. 最后做项目实战

**真正能让你完成转型的，不是看完多少篇文档，而是做出多少能讲清楚的项目。**
