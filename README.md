# Agent Fullstack Learning

> 一套面向 **iOS 开发工程师转 Agent 全栈工程师** 的学习资料仓库。  
> 目标不是只告诉你“学什么”，而是尽量帮你做到：**看得懂、学得会、查得到、练得动、做得出项目。**

</p>

## ✨ 这个仓库适合谁

如果你符合下面几点，这套资料会很适合你：

- 会 **Objective-C / Swift**
- 有 **iOS 客户端开发经验**
- 大学学过 **Java**，但后端实战偏少
- 想补 **前端、Java 后端、Agent 工程能力**
- 希望从“会调模型 API”进阶到“能独立落地 Agent 产品”

这套资料更偏向：

- **先做出东西，再逐步补系统性**
- **先能实战，再逐步补深度**
- **围绕项目学，而不是只背概念**

## 📌 学习建议

给你一个最实用的建议：

**不要把这些资料当收藏夹，要把它们当施工图。**

比较合理的方式是：

1. 先读一份总览
2. 再读一份教学文档
3. 一边做练习一边翻速查手册
4. 最后做项目实战

真正能让你完成转型的，不是看完多少篇文档，而是：

- 做出多少能运行的东西
- 讲清楚多少模块为什么这么设计
- 能不能把知识串成一个完整系统
