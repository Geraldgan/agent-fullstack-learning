# Agent Fullstack Learning

这是一套面向 iOS 开发转 Agent 全栈工程师的学习资料。

## 内容结构

### 01-roadmap
- 路线图
- 计划
- 项目方案
- 面试知识点

### 02-teaching
- 一周速成教学版
- 三个月系统教学版
- 前端系统教材
- Java 后端系统教材

### 03-reference
- 前端速查手册
- Java 后端速查手册
- 详细知识点大全

### 04-practice
- 前端练习题与答案版
- Java 后端练习题与答案版
- 项目骨架设计说明

## 使用建议

1. 先看一周速成教学版，建立整体认知
2. 再看前端 / Java 系统教材，补齐细节
3. 配合速查手册边学边查
4. 再做练习题和项目实战
